🚗 Road Accident Analysis Dashboard

Project Overview

Developed an interactive **Power BI dashboard** to analyze road accident data and identify key trends, patterns, and factors affecting road safety.

Tools & Technologies
* Power BI
* Power Query
* DAX
* Data Modeling

Key Features
* Year-over-Year accident trend analysis
* Vehicle-wise accident analysis
* Weather and road condition insights
* Interactive filters and KPI dashboards
* Casualty and accident pattern visualization

Outcome
Generated actionable insights to support data-driven decision-making for road safety while strengthening skills in **Power BI, DAX, data cleaning, and data visualization**.
